Your Own Snake Game:
                           >> HOW TO PLAY <<
--------------------------------------------------------------------------
1. Download the complete files.
2. store it in any folder or on your desktop.
3. Double click on INDEX.html.
4. Enjoy Your Snake Game.  


                             >> Levels <<
--------------------------------------------------------------------------
Level 1. =>normal movements catching food
Level 2. =>Increase In snake Speed after score reaches 30pts
Level 3. =>Jumping Food


                          >>>  CONTROLS  <<<
---------------------------------------------------------------------------
Arrow up,down,left,right :Snake movement Control
q    :quit game
s    :save game and exit
1    :load saved game
spaceBar :To pause game
n    :for new game

                            >>  SCORE  <<
--------------------------------------------------------------------------
1 score per 1 food Eat

                          >> REQUIREMENTS <<
---------------------------------------------------------------------------
Any Browser above Internet Explorer 8.
